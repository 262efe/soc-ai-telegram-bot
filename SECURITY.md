# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue.

Instead, contact me directly:
- 📧 Email: hi@efealtintas.com

I will respond within 48 hours and work on a fix as soon as possible.
Thank you for helping keep this project secure.

Bunun yerine doğrudan benimle iletişime geçin:
- 📧 E-posta: hi@efealtintas.com

48 saat içinde yanıt vereceğim ve mümkün olan en kısa sürede düzeltme yapacağım.
Projeyi güvenli tutmaya yardımcı olduğunuz için teşekkürler.
